# ZX Sound Detector

A Firefox extension that helps you manage and control media playback across all your tabs from one convenient location. Never lose track of which tab is playing sound again!

## Features

- **Global Media Control**: View and control all media playback from one place
- **Tab Management**: See all tabs with active or recently played media
- **Playback Controls**: Play, pause, skip tracks, and mute directly from the popup
- **Smart History**: Recently played tabs stay in the list for quick access
- **Modern UI**: Clean, intuitive interface with smooth animations
- **Lightweight**: Minimal performance impact on your browser

## Installation

### From Firefox Add-ons

1. Visit the [Firefox Add-ons page](https://addons.mozilla.org/) (coming soon)
2. Click "Add to Firefox"
3. Confirm the installation

### Manual Installation

1. Download or clone this repository
2. Open Firefox and type `about:debugging` in the address bar
3. Click "This Firefox" in the left sidebar
4. Click "Load Temporary Add-on"
5. Select any file from the extension directory

## How to Use

1. Click the ZX Sound Detector icon in your toolbar
2. View all tabs with active or recent media playback
3. Use the controls to:
   - Play/Pause media
   - Skip tracks (when supported)
   - Mute/unmute tabs
4. Click on a tab's title to switch to it
